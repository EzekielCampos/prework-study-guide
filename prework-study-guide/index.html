Prework Study Guide

HTML

  • The head elements contains information about the webpage that is used in the background
  • The body elements is information that the user will be able to see and interact with.
  • Meta elements are different specifications for the data to specify things like character set, keywords, etc.

CSS

  • There are three different ways to add css components to a html file, the most intuitive way is to use an external style sheet
  • CSS deals with the specific design and layout of a webpage like the colors, margins, and borders
  • Class attributes allows add styles to certain elements. These classes can be declared in the sections in the html file

Git

  • Git is a version control system software that allows people and teams to collaborate on projects together
  • Keeps track of different versions of code and allows others to review team members source code that can be implemented to a main program
  • Individual programmers can work from their own computer and upload the files to the team's repository
  • Branches are derived from the main branch to write code that won't affect the main branch until it is reviewed and approved to be merged
